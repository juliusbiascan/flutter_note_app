
# Flutter Note App

This is Simple Note App using Flutter (bloc, http ) front end, Nodejs (Typescript, Nodejs, Express, Prisma) backend.


## Tech Stack

**Client:** Flutter

**Server:** Node, Express, Typescript , Prisma

## Run Locally

Clone the project

```bash
  git clone https://link-to-project
```

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

```bash
  npm start
```
